# Space Demo Plan

## Objetivo

Demonstrar o Omni ajustado com LoRA em uma interface simples e segura.

## Escopo mínimo

- caixa de input
- resposta do modelo
- exibição opcional de OIL gerada
- aviso de limitações

## Componentes

- Gradio ou Streamlit
- carregamento do modelo base + adapter
- prompts de exemplo do domínio Omni

## Recomendações

- manter exemplos curtos
- mostrar que é uma demo de adapter, não o runtime completo
- destacar que governança do runtime principal não está totalmente reproduzida no Space

## Próximos passos

- exibir task family prevista
- comparar saída base vs adapter
- anexar links para dataset card e model card
