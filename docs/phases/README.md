# Phase documentation

## How to read this folder

- **Phases 30 and below** (`phase-30.md`, material under `docs/archived_phases/`) are **historical closure** and early convergence records. They remain for auditability; they are not the live definition of the product.
- **Phases 31–40** are the **documented cognitive runtime maturation path** implemented in code today: reasoning, memory intelligence, planning, learning, strategy, performance, coordination, decomposition, controlled evolution, and governed continuous improvement.
- **Phase 40** is the **current documented maturity band** for the self-improving layer (governed improvement orchestration on top of Phase 39). There is no “Phase 41” in-repo scope.
