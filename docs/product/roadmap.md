# Roadmap (detailed)

This document expands the condensed table in the root `ROADMAP.md`. **Phase 40 is the current documented maturity target** for the governed self-improvement layer.

## Phases 31–40 (implemented documentation bands)

| Phase | Focus |
|------|--------|
| 31 | Cognitive reasoning layer (`ReasoningEngine`, OIL handoff) |
| 32 | Memory intelligence layer (unified memory / retrieval planning) |
| 33 | Planning intelligence (planning engines, execution-ready plans) |
| 34 | Runtime learning (learning engine, records, traces) |
| 35 | Strategy adaptation (strategy engine, adaptation payloads) |
| 36 | Performance optimization (compression/caching boundaries) |
| 37 | Multi-agent coordination (coordinator traces, readiness) |
| 38 | Task decomposition (structured subtasks, linkage to plans/reasoning) |
| 39 | Controlled self-evolution (governed proposals, validation, optional apply, rollback) |
| 40 | Governed self-improving system (simulation, approval, staged rollout, monitoring) |

Authoritative per-phase notes: `docs/phases/phase-31.md` … `phase-40.md`.

## Phase 30.x (historical)

Phase 30 documentation (`docs/phases/phase-30.md` and archived reports) captures **governance convergence** and early evolution-control closure — preserved for history, not the live feature definition of phases 31–40.

## Beyond Phase 40

There is **no Phase 41** scoped in this repository. Future industrial work would extend operator UX, statistical monitoring, and promotion workflows — still without unconstrained autonomous engineering.
